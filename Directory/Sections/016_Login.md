# Login - 登陆界面
1. 自动登陆
    + 读取本地数据
    + 网络数据校验
    + 进入游戏
2. 快捷登陆
    + 获取设备信息
        - UUID等
    + 获取第三方凭证
        - 微信等
    + 获取身份信息
        - 人脸信息等
3. 文本键入登陆
    + 手机验证码登陆
    + 账号密码登陆
4. 密码找回
    + 验证码
    + 验证问题
    + 申诉
        - 实名信息
        - 注册信息
        - 使用信息
4. 注册入口
    + 引导用户
5. 注册界面
    + 手机注册
    + 邮箱注册
    + 账号注册
6. 实名认证
    + 法规需求
6. 入口
    + 自动进入
    + 点击进入
