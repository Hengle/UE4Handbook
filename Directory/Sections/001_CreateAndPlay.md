# Create And Play - 创建并游玩
1. 创建模板工程

    + 动态添加模板内容
2. 试玩工程
    + Viewport
        + Simulate
        + New Window
        + Standalone
        + Mobile Preview
        + 快捷键
            - F1(线框)
            - F2(无光照)
            - F3(光照)
            - F4(仅光照)
            - F5(性能)
            - F8(控制/弹出)
            - F9(截屏)
        + 联网测试(多玩家)
            - 用户端数目(Number of Clients)
            - 专用服务器(Dedicated Server)

4. 添加商城资源
    + 获取

5. 添加其它资源
