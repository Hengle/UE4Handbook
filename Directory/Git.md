### git的（表面）原理
+ 多个看不见的仓库
+ 合理的工作流
+ index管理的暂存区
+ HEAD指向的当前分支版本的引用
+ objects哈希目录结构
+ refs数据引用
+ and so on

+ repository
+ remote repository
+ stash
+ submodule
+ suptree
