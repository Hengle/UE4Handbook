# UE4Handbook
    这是个学习手册，用于记录UE4学习历程,以及其它方向的杂谈。
    本人是游戏行业从业者，才疏学浅，项目中错误、遗漏、可优化部分，望众大佬提出指正，我会虚心接受。
    学习过程中，我会直播或录制视频，以供后来者参考借鉴。
    视频中有错误及遗漏会不定期修正，重录或添加修正说明。
    视频中存在大量通俗说法，严谨请移步官方教学视频。
